CSS Cheat Sheet

Border

The border represents the representation of an element's bounding box.

Border-left, border-right, border-top, border-bottom

The border-left, the border-right, the border-top and the border-bottom are partial implementations of an element's bounding box.

Border-radius

This determines the curvature of the bounding box of the element. Effectively, it morphs the rectangle into one with rounded corners.

Color

This alters the color of the text contained within an element.

Font

This alters the font attribiutes of the text contaied within an element. This is a shorthand for changing all of the different font attributes at the same time.

Font-family

This alters the font family as a whole of all different kinds of text.

Font-style, font-weight

This defines the specific font style to be used from its chosen font family or weighed differently. Could be styled or weighed.

Width, Height

These control de dimensions of the element's bounding box. This way, elements can be made narrower, wider or shorter or taller. Borders shown for clarity.

Margin

This sets the distance an element has with its neighbors. Can be set independently for each direction. Top, right, bottom, left. This paragraph has a uniform margin which applies to all sides. Border shown for clarity.

Padding

Similarly to margin, it creates spacing around an element. BUT this time, it's actually creating spacing between the content of an element, and its bounding box, not between the element's bounding box and the elements around it. Border shown for clarity.

Overflow

This determines the way an element's content is displayed when it doesn't fit within the element's padding. This can be done plenty of ways, but in this example it's chosen to show the overflow.
Which may overlap with adjacent elements. Box shown for clarity.

Display

Defines how an element and its children will be treated. Either as blocks, inlne-blocks, a grid, for example. This one uses inline in each to make inline headers and paragraphs.

Position

Defines where in the screen an element is located. This one should stick to the bottom.
