p - Paragraph

A paragraph provides a space where text can be written in block-style, such that two paragraphs are not displayed next to one another, and are instead displayed below one another sequentially according to the order they were written in, in the <body> tag of the .html file.

A <div> can simply be thought of as a block-level element grouping which provides structure, without any semantic meaning in the document. In it, one can group different elements as one single element.

strong - Strong Text

The <strong> is an inline-level element which makes bold text. With the peculiarity that it has semantic significance, which entails strong importance to the text.

i - Italize

The <i> is an inline-level element which italizes text, with the purpose of having an alternative voice or tone.

u - Underline

The <u> is an inline-level element which underlines text.

b - Bold Text

The <b> is an inline-level element which bolds text. To stylistically offset the text.

strike - Strike text

The <strike> is an inline-level elemnt which creates strike-through text. Useful for korections corrections.

small - Small text

The <small> is an inline-level element which shrinks text such that it appears to have a smaller size.

em - Emphasize text

The <em> is an inline-level element which italizes text with the intent of emphasizing it.

header - Header

The <header> is a block-level element which defines the content of the page which will be displayed at the very top of...well...a something, which can be any semantically significant structure which contains it.
