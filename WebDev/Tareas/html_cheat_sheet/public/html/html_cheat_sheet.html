HTML-Cheat-Sheet

a - Anchor

An anchor provides a link to a page or resource. By using the href attribute, one can specify either a relative or absolute path for the page or resource the anchor links with. An example would be a link to the TEC page which sells the tickets for LiFE's upcoming event: VESUS Divxs, in which I will partake in on May 28 as a lead singer. Link a los boletos.

p - Paragraph

A paragraph provides a space where text can be written in block-style, such that two paragraphs are not displayed next to one another, and are instead displayed below one another sequentially according to the order they were written in, in the \ tag of the html file.
